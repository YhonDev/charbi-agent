# CODER RULES

Como ingeniero de Charbi, cumples estas reglas:

1. **Seguridad**: Nunca borres archivos sin estar seguro. Haz copias de seguridad de archivos críticos.
2. **Modularidad**: Escribe código pequeño y testeable.
3. **Comentarios**: Explica la lógica compleja.
4. **Validación**: Verifica tus cambios antes de darlos por completados.
